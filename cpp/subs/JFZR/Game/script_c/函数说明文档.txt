邮寄金币给某个角色("填你要邮寄的角色名", 10000, 10000);
第一个参数是你要邮寄的角色名,第二个参数是自己要保留的金币数量
身上的金币减去第二个参数，得到的结果大于第三个参数的数值，才会邮寄


交易物品过滤，可以过滤三种，装备栏、材料栏、消耗品栏，分别对应三个脚本命令：添加消耗品的交易物品过滤器、添加装备的交易物品过滤器、添加材料的交易物品过滤器。
添加消耗品的交易物品过滤器({
		{ "下级鉴定书" , 空, 空, 以指定的价格拍卖(33, 1) },
		{ "肉脯干" , 空, 空, 以指定的价格拍卖(30, 655, 1, 2) },
		{ "刚出炉的面包" , 空, 空, 按市场价的某个范围拍卖(40, 3, 2, 2) },
		{ "中级鉴定书" , 空, 空, 按市场价的某个范围拍卖(20, 25000, 1, 1, 2) },
		{ "刚出炉的面包" , 空, 空, 邮寄物品("张三") },
	});
添加过滤的话，就像上面这样。
{ "下级鉴定书" , 空, 空, 以指定的价格拍卖(33, 1) }，第一个参数是物品的名称，第二个参数是物品品质(1：低级，2：中级，3：高级，4：稀有，5：传说)，第三个参数是物品等级，第四个参数是物品的处理方式。
物品名称、物品品质和物品等级不能同时为空，物品处理方式一定不能为空。
物品名称、物品品质与物品等级，这三个条件用来过滤物品，如果满足条件，就会按照提供的方式去处理该类物品。
比如添加这样一条物品过滤：{ 空 , 4, 50, 邮寄物品("李四") }，表达的意思是把稀有品质以上(包含稀有品质)，并且物品等级大于等于50级的物品，全都邮寄给李四。
物品的处理方式分为两大类：邮寄和拍卖。
拍卖的处理方式比较细致，分为两类。
以指定的价格拍卖(30, 655, 1, 2)：
第一个参数表示拍卖的价格，第二个参数表示要拍卖的物品数量(可以为空，为空即取默认值，默认为1)，第三个参数表示拍卖条数限制(可为空)，
第四个参数表示帐号类型(1:普通号，你的普通号将会购买你拍卖的这个物品，2：仓库号，你的仓库号将会购买你拍卖的这个物品，3：中转站，你的中转站将会购买你拍卖的这个物品。可以为空，为空即取默认值，默认表示你拍卖的这个物品，其他号都不会去购买，意思是只会卖给玩家)。

按市场价的某个范围拍卖(20, 25000, 1, 1, 2)
第一个参数表示市场价格范围，若是市场价为100，20表示将以120的价格拍卖此物品，-20表示将以80的价格拍卖此物品
第二个参数表示默认的价格，获取市场价格并不总是成功的，若是失败了，就会使用这个参数表示的价格来拍卖此物品了
第三个参数表示拍卖的数量，可以为空，默认为1
第四个参数表示拍卖条数限制，可为空
第五个参数表示帐号类型，其意义参考以指定的价格拍卖
